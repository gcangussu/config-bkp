# Ubuntu
## Change mouse button modifier to `Super`

This will enable you to use `Alt` + `Click` for multi cursors in VSCode.

```bash
dconf write /org/gnome/desktop/wm/preferences/mouse-button-modifier '"<Super>"'
```
